# 🎴 Pixel Tarot v3.0

A pixel-style tarot reading web app with AI interpretation and animation effects.

---

## ✨ Features

- 🎴 Tarot card draw system based on birth date + gender
- 🧠 Deterministic random algorithm (same input = same result)
- 🤖 AI interpretation system (fallback supported)
- 🎬 3D card flip animation
- 📱 Fully responsive (mobile + desktop)
- 🎨 Pixel + modern UI hybrid (Canva-style design)
- 🖼️ Tarot card image support
- 💾 Local stateless architecture (no backend required)

## 🧠 Core Concept

This project simulates a “personality-based tarot system”:
birthdate + gender → hash → deterministic shuffle → tarot cards

No real randomness → ensures reproducibility.

## 🎴 Example Flow

1. User enters birth date & gender
2. System generates seed
3. Tarot deck is shuffled deterministically
4. 3 cards are drawn
5. Cards flip with animation
6. AI interpretation is generated
